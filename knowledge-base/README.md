# 지식베이스 (Knowledge Base)

AI가 부동산 투자 질문에 답하기 전에 먼저 검색해야 하는 근거 저장소. 설계 배경은 `docs/강의서문.md` 부록 B 참고.

## 사용 원칙

1. **질문 → 검색 → 근거 인용 → 답변** 순서를 지킬 것. 근거 없이 추측하는 답변은 반려.
2. 정책·세금처럼 자주 바뀌는 항목은 "현재 무엇인가"뿐 아니라 **시행일·해제일·근거·영향·출처**를 함께 기록.
3. 출처가 2차 자료(뉴스 등)뿐이면 "1차 출처 확인 필요"라고 명시하고, 확정된 사실처럼 쓰지 않는다.
4. 만료일이 있는 규제(예: 토지거래허가구역 재지정)는 만료 전에 반드시 재검색해서 갱신한다.

## 폴더 구성

| 폴더 | 내용 |
|------|------|
| `00_변경이력` | 정책·세법·API·프롭테크 서비스의 변경 이력 모음 |
| `01_정책규제` | 조정대상지역, 투기과열지구, 토지거래허가구역, 대출규제(LTV·DTI·DSR) 등 |
| `02_세금` | 취득세·양도세·종부세 등, 주택수·보유기간·조정지역 여부별 조건 |
| `03_법령` | 민법, 민사집행법, 주택임대차보호법, 도시정비법 등 원문/요약 |
| `04_판례` | 명도, 유치권, 법정지상권, 대항력, 최우선변제 관련 판례 |
| `05_실무QA` | 실전 상담 질문과 답 |
| `06_체크리스트` | 경매·재개발 등 실행 전 확인 항목 |
| `07_공식데이터` | 국토부 실거래, 법원경매, 공공데이터포털 등 원본 데이터/수집 스크립트 |
| `08_지역개발계획` | GTX·재건축 등 지역별 개발계획 |
| `09_AI오답노트` | AI가 틀렸던 사례와 정정 기록 |
| `10_나의투자원칙` | 개인 투자 기준과 철학 |

현재 `01_정책규제`만 실제 내용이 채워져 있고(2026-07-08 기준), 나머지는 폴더 구조만 잡아둔 상태입니다.

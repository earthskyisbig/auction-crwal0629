# 상가건물 임대차보호법 — 경매·투자 관점 핵심 조문

> 마지막 확인일: 2026-07-08 (웹 검색 기반 정리 — 실제 사건 적용 전 국가법령정보센터 law.go.kr 원문 재확인 필수)

## 1. 환산보증금 기준 (적용범위)

- 개념: 보증금 + (월세 × 100)으로 환산한 금액. 이 금액이 지역별 기준 이하여야 법의 전면 적용(우선변제권 등)을 받음.
- 2026년 기준(검색 시점 확인분 — 시행령 개정 시 변동 가능, 재확인 필요):
  - 서울: 9억 원
  - 과밀억제권역·부산: 6억 9,000만 원
  - 광역시 등: 5억 4,000만 원
  - 그 밖의 지역: 3억 7,000만 원
- 출처: 검색 결과 종합(on-estate.com 등) — law.go.kr 상가건물 임대차보호법 시행령 원문 대조 권장

## 2. 대항력 · 우선변제권

- 대항력: 임차인이 사업자등록 신청 + 건물 인도를 마치면 그 다음 날 0시부터 대항력 발생.
- 우선변제권: 대항요건 + 확정일자를 갖추면 경매·공매 환가대금에서 후순위권리자보다 우선 변제.
- 최우선변제권(소액임차인): 대항요건만 갖추면 발생하며, 경매 시 일정액을 다른 권리자보다 최우선 배당.
- 출처: https://easylaw.go.kr/CSP/CnpClsMain.laf?popMenu=ov&csmSeq=627&ccfNo=4&cciNo=2&cnpClsNo=2

## 3. 환산보증금 초과 시에도 보호되는 3가지

- 환산보증금이 기준을 넘어 법의 전면 적용 대상이 아니어도 다음은 보호됨:
  1. 계약갱신요구권 (최대 10년)
  2. 권리금 회수 기회 보호
  3. 대항력 (건물주가 바뀌어도 유지)
- 출처: 검색 결과 종합(hankyung.com, smatch.kr)

## 4. 권리금 보호

- 조문 요지: 임대인은 임대차 종료 6개월 전부터 종료 시까지, 임차인이 주선한 신규임차인이 되려는 자로부터 권리금을 지급받는 것을 방해해서는 안 됨.
- 영향: 경매로 소유권이 이전되는 경우 권리금 보호 규정이 낙찰자에게 그대로 승계되는지는 개별 사안(계약 종료 사유, 낙찰자의 지위)에 따라 달라짐 — **원문·판례 대조 필요**
- 출처: 검색 결과 종합
